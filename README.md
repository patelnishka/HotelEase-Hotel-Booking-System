# HotelEase - Hotel Booking System

A full-stack hotel booking web application developed using PHP, MySQL, HTML, CSS, JavaScript, PHPMailer, and DomPDF. The system enables users to search and book hotel rooms online, manage reservations, make payments, generate invoices, and submit reviews, while administrators can manage rooms, bookings, users, and reports through a dedicated dashboard.

---

## Features

### User Features

* User Registration and Login
* Forgot Password & Password Reset via Email
* Browse Available Rooms
* Advanced Room Search and Filtering
* Hotel Reviews and Ratings
* Online Room Booking
* Online Payment Processing
* Booking History Management
* PDF Invoice Generation
* Email Notifications

### Admin Features

* Admin Dashboard
* Room Management

## Installation Guide

### Prerequisites

* XAMPP
* PHP
* MySQL
* Web Browser

### Steps

1. Clone the repository

```bash
git clone https://github.com/patelnishka/HotelEase-Hotel-Booking-System.git
```

2. Move the project folder to:

```text
xampp/htdocs/
```

3. Start Apache and MySQL using XAMPP.

4. Create a database named:

```text
hotel_ease_db
```

5. Import the SQL file from the database folder.

6. Configure database settings in:

```text
config/db_config.php
```

7. Open the application in your browser:

```text
http://localhost/hotel_ease_bs
```
